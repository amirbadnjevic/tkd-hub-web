The Story Behind TKD Hub

Hi, I'm Amir - a software engineer and Taekwondo parent based in Sweden. I built TKD Hub to solve a problem I ran into every week.

Finding Taekwondo events meant jumping between too many websites that were hard to use, especially on mobile. As a parent of a young athlete, keeping track of upcoming events quickly became a weekly headache.

So I built a better way - so athletes, parents, and coaches can spend less time searching and more time focusing on training.

"This is a passion project built for the Taekwondo community and inspired by my daughter's journey in the sport."
